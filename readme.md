## Usage

Data is formated as follows:
```json
{REGION : {PROVINCES: [...],
           registered_voters: 123,
           PROVINCIAL_DATA: {PROVINCE :
                                 {BATCH# :
                                      {location_code: ...,
                                       result: [...]
                                      }
                                  }
                             }
          }
} 
```
_For more examples, please refer to the [Documentation](https://example.com)_
